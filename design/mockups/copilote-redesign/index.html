Convenu

A — Focus

Chat épuré centré (Linear / ChatGPT). Tools et raisonnement vivent dans le flux, repliables mais ouverts par défaut. Assistant en ghost (sans cadre) pour un rendu éditorial. Le plus sobre, le plus lisible — l'option sécurité.

Réorganisé

B — Duality

Split 2 colonnes : conversation clean à gauche, inspecteur à droite (tools détaillés, raisonnement complet, fil chronologique). La lecture et la profondeur se séparant — l'option pro / power user.

Novateur

C — Command

Command center 3 zones : rail contexte supply (OF, stock, CTP de l'article actif) à gauche, conversation au centre, dock capacités à droite. Source chips cliquables qui pointent vers le contexte. Le copilote ancré dans le métier — l'option immersif.

Source de vérité tokens : inertia-react/styles/app.css (scope .theme-airbnb) · Page actuelle : inertia-react/pages/copilote.tsx (chat sans Bubble).
Bubble — mapping variants : default = user (Rausch, align end) · ghost = assistant (sans cadre, full width) · outline/muted = tool traces · destructive = erreur.
Décisions transverses : statuts OF conservés (ferme green · planifie Babu teal · suggere Arches) · focus = ink #222 pas Rausch · une seule ombre (hover float) · pas de dark mode.
Prochaine étape : choisir une direction (ou mixer, ex. « A + rail contexte de C »), puis porter en composant React Bubble + adapter copilote.tsx.
